Vscode开发qt Cmake

Posted on 2025-04-14 • 17 words • 1 minute read
Last updated on 2025-04-14

新建一个cmake qt项目，看构建步骤
cmake configure cmake build cmake run

对应到vscode，第一步，设置c++的智能补全，这个在qmake那篇文章中说了
第二步，安装cmake tools、cmake插件，配置这两项 然后在CmakeList.txt文档中加"set(CMAKE_PREFIX_PATH “D:/qt6.8.2/6.8.2/mingw_64”)"，目的是cmake编译时能找到qt相关的库

ctrl+shift+p，分别执行cmake configure、cmake build、cmake run，完成
